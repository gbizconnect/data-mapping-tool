# Data Mapping Tool Frontend Module.


## 👨‍💻 Description

Simple whiteboard to reproduce the basic functionality of FigmaJam



## 🏃 Run Locally (develop & debug)


```
git clone 
npm install -g yarn
yarn install
```

```
yarn workspace frontend dev
```

0.0.0.0でLISTENする場合
```
yarn workspace frontend dev --host 0.0.0.
```
